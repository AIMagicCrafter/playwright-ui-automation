## Playwright UI Automation

This repository contains UI automation tests built using Playwright (JavaScript),
focused on clean design, stable locators, and realistic user workflows.

### Application Under Test
- https://demo.playwright.dev/todomvc

### Test Structure
- **Smoke tests**: Critical user flows
- **Regression tests**: State-based behavior validation

### Tech Stack
- Playwright
- JavaScript
- Node.js

### How to Run
```bash
npm install
npx playwright test
